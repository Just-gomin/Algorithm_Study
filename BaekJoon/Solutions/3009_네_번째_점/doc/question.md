# 문제 해결 단서

1. 세 점이 주어졌을 때, 축에 평행한 직사각형을 만들기 위해서 필요한 네 번째 점을 찾습니다.

<br><br>

# 문제 해결 방법

<br>

## Python

1. x좌표와 y좌표를 담는 리스트를 만듭니다.
2. 좌표가 입력될 때 마다 x좌표는 x좌표 리스트에 있다면 해당 x좌표를 제거하고, 없다면 x좌표 리스트에 담습니다.
3. y좌표 또한 x좌표를 취급한 것과 같이 행동합니다.
4. 리스트들에 남아있는 좌표들을 출력합니다.
