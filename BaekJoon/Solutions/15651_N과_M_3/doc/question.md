# 문제 해결 단서

1. 자연수 N과 M이 주어집니다.
2. 1부터 N까지 자연수 중에서 중복 가능하게 M개를 고른 수열을 출력합니다.
3. 한 줄에 하나씩 문제의 조건을 만족하는 수열을 출력합니다.

<br><br>

# 문제 해결 방법

<br>

## Python

1. 1부터 N까지의 자연수를 M개씩 넣은 리스트를 생성합니다.
2. 숫자 리스트의 데카르트 곱을 구합니다.
3. 결과를 한줄에 하나씩 출력합니다.
