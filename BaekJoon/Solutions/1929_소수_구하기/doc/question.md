# 문제 해결 단서

1. 두 수 M, N이 주어졌을 때, M이상 N이하의 소수를 모두 출력하는 프로그램을 작성합니다. ( 1 <= M <= N <= 1,000,000)
2. 한 줄에 하나씩 증가하는 순서대로 소수를 출력합니다.

<br><br>

# 문제 해결 방법

<br>

## Python

1. 범위가 주어지므로, N까지의 모든 소수를 에라토스네스의 체를 이용해 찾아냅니다.
2. 찾아낸 소수들 중 M과 N사이에 있는 모든 수를 출력합니다.
