# 문제 해결 단서

1. 2차원 좌표 N개가 주어집니다.
2. 주어진 좌표를 y좌표가 증가하는 순으로, y좌표가 같다면 x좌표가 증가하는 순서로 정렬시킵니다.
3. 정렬된 좌표를 한줄에 하나씩 출력합니다.

<br><br>

# 문제 해결 방법

<br>

## Python

1. n개의 좌표를 입력받아 list에 저장합니다.
2. 좌표 리스트를 sort 함수를 이용해 정렬합니다.
3. sort 함수의 key를 lambda 함수를 이용해 y좌표, x좌표 순으로 비교시킵니다.
4. 정렬된 리스트를 출력합니다.
