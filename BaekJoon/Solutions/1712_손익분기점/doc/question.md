# 문제 해결 단서

1. 고정비용 A, 가변비용 B, 판매가격 C가 주어집니다. A, B, C는 21억 이하의 자연수입니다.
2. A, B, C를 이용해 최초로 수입이 총 생산 비용보다 많아져 이익이 발생하는 지점(손익분기점)을 찾아 출력합니다.
3. 손익분기점이 존재하지 않는다면 -1을 출력합니다.

<br><br>

# 문제 해결 방법

<br>

## Python

1. A, B, C를 입력받습니다.
2. 수익 = C*n - (A + B*n)
3. 수익이 점차 커진다면 손익분기점을 구해 출력합니다.
4. 수익이 커지지 않는다면 -1을 출력합니다.
