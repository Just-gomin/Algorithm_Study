# 문제 해결 단서

1. 두 정수 A,B가 입력됩니다.
2. A와 B의 대소 비교를 진행하고 결과를 출력합니다.

<br><br>

# 문제 해결 방법

<br>

## Python

1. input과 split을 이용해 값을 입력 받습니다.
2. map과 int를 이용해 입력 값을 정수로 변환합니다.
3. if와 elif, else를 이용해 A,B의 대소를 비교하여 출력합니다.
