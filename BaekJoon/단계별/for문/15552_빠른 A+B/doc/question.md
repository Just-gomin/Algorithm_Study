# 문제 해결 단서

1. 테스트 케이스의 개수 T가 주어지며 T는 최대 1,000,000 입니다.
2. 다음 T줄에는 두 정수 A와 B가 주어집니다. 1 <= A,B <= 1000
3. A와 B를 더한 값을 출력합니다.

<br><br>

# 문제 해결 방법

<br>

## Python

1. input 대신 sys.stdin.readline을 이용해 빠른 입력을 받습니다.
2. split과 int를 사용하여 입력을 정수형으로 변환합니다.
3. 덧셈 결과를 출력합니다.
