## 📌 Commit Message

`.gitmessage.txt`을 참고해 커밋 메세지를 작성한다.

커밋 시 자동으로 템플릿을 불러오게 하려면 프로젝트의 루트 경로에서 다음의 명령을 수행한다.

```shell
git config commit.template .gitmessage.txt
```
